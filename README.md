## 📖 About

**JR Addons** is a custom-built Elementor plugin that extends Elementor's capabilities with WooCommerce-focused widgets. Built with performance, customization, and user experience in mind.

---

## ✨ Features

- 🎨 **Fully Customizable** — Every widget comes with extensive style controls
- ⚡ **AJAX Powered** — Smooth user experience without page reloads
- 📱 **Fully Responsive** — Mobile, tablet, and desktop optimized
- 🛒 **WooCommerce Ready** — Deep integration with WooCommerce
- 🎯 **Lightweight** — Optimized code for fast loading
- 🌍 **Translation Ready** — i18n support included
- 🎭 **Modern Design** — Clean, professional UI/UX

---

## 🧩 Widgets

### 🛍️ WooCommerce Widgets

| Widget | Description |
|--------|-------------|
| 🔍 **Search Widget** | AJAX live search with product preview |
| 🎬 **Slider Widget** | Image & Testimonial slider modes |
| 🛒 **Header Icons** | Cart, Wishlist, Account, Order Tracking icons |
| 📦 **Mini Cart** | Hover dropdown + Side drawer mini cart |
| 🔥 **Top Selling Products** | Display best-selling products |
| 🎠 **Product Carousel** | Customizable product carousel |
| ⚡ **Add to Cart Effects** | Spinner + Tick + Toast notifications |
| 🎯 **Product Action Buttons** | Add to Cart, Buy Now, WhatsApp, Call (4 buttons, 3 layouts) |
| 💰 **Product Price** | Sale/Regular/Old price with custom currency support |
| ⭐ **Product Reviews** | Complete review system with replies, avatars, badges |
| 📝 **Short Description** | Word limit with Read More toggle |
